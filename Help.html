Localization Spreadsheet JSONify

To use this app:

First, create a localization spreadsheet that looks like this one.

Simply call  
with the following querystring parameters set:

sheet_id         required   ex: 0AqrUvD5TZZs3dF9ULUh5X1JlakVJRGFHaWRZQmFuZEE
sheet_name       required   ex: Main
threshold        optional   ex: 0.8

sheet_id

    The key parameter from a Google Spreadsheet URL, e.g. 
    "https://docs.google.com/spreadsheet/ccc?key=0AqrUvD5TZZs3dF9ULUh5X1JlakVJRGFHaWRZQmFuZEE"

sheet_name

    The sheet name to use inside of the spreadsheet.

threshold

    Controls the percentage of fully-translated strings a translation must have
    before it is deployable. In other words, if you have 10 total strings, and
    your threshold is 0.8, then at least 8 of the strings in a translation must
    be set, or the language is not considered deployable.


Make sure the spreadsheet you want to access is public or that you are logged in 
and have access to it.
